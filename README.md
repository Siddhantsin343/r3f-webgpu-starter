# r3f-webgpu-starter

A minimal starter template for building 3D web applications with **React Three Fiber** and **Three.js WebGPU Renderer**.

## Features

* React + Vite
* Three.js
* React Three Fiber (R3F)
* Drei helpers
* WebGPU Renderer setup
* Example cube
* Clean project structure

## Installation

```bash
npm install
```

Or install dependencies manually:

```bash
npm install three @react-three/fiber @react-three/drei
```

## Run

```bash
npm run dev
```

## Why?

Setting up WebGPU with React Three Fiber can be repetitive. This starter provides a minimal boilerplate so you can start building immediately.
